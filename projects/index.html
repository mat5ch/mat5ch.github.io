Melody Helpr

Melody Helpr is an app that tries to help with creating melodies from scratch. It is meant to be used in companion with the Open Source recording software Ardour. You can create new melody sequences, double or halve them, recreate at any point and finally transfer the created sequence from Melody Helpr back into Ardour to further work with it.

Find this project at my GitHub.
